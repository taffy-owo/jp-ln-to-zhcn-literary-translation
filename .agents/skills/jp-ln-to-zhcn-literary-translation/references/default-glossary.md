## Honorifics and Address

| Japanese | Default handling | Note |
|---|---|---|
| さん | 姓/名 + 同学/先生/小姐/老师/直接称名/省略 | 编码礼貌和距离，别默认“桑”。 |
| くん | 名字 / 同学 / 你 / 君（极少） | 看年龄、校园、上下级、暧昧感。 |
| ちゃん | 小 + 名 / 名字 / 亲昵称呼 / 省略 | 保留亲昵、幼态、揶揄或居高临下。 |
| 様 | 大人 / 阁下 / 殿下 / 姓名+称谓 / 您 | 幻想/宗教/政治/服务业要分开。 |
| 先輩 | 学长 / 学姐 / 前辈 | 校园优先学长/学姐；职场可前辈。 |
| 先生 | 老师 / 医生 / 先生 / 老师称呼 | 依职业。 |
| お兄ちゃん | 哥哥 / 哥 / 兄长（特殊风格） | 记录角色专属称呼。 |
| お姉ちゃん | 姐姐 / 姐 | 姐妹关系线要稳定。 |
