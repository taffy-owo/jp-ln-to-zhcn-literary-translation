# 连接词、转折词与段落节奏控制

此文件用于解决日轻译文里常见的“可是/不过/即便如此/就在这时/那一瞬间”反复出现的问题。目标不是把连接词全部删掉，而是让中文段落靠动作、语气、因果和换行自然推进。

## 核心判断
日文常用 `でも / だけど / しかし / それでも / とはいえ / その時 / 次の瞬間` 明示逻辑；中文小说不一定需要逐个显性标出来。翻译时先判断它的功能，再决定是否保留。

常见功能：
- 逆接：前后事实相反。
- 让步：知道如此，但人物仍然这么做。
- 补充限制：前句成立，但有一个小例外。
- 场景推进：同一时间发生新动作。
- 情绪转折：人物心理突然改变。
- 叙述节拍：只是日文句首垫步，中文可省。

## 默认规则
1. 不要把每个 `でも/だけど/しかし` 都译成“可是/不过/但是”。
2. 同一章里同一个强转折词不要高频复用；相邻段落尤其不要连续以“可是/不过/即便如此/就在这时”开头。
3. 能靠动词或人物动作表达关系时，优先删掉连接词。
4. 能把“转折”压进主语和谓语时，不单独加“可是”。例如“可是她没有回答”常可改为“她没有回答”。
5. “即便如此/尽管如此”只在需要庄重让步、情绪坚持、叙述强调时使用；普通场景用“还是/仍然/却/只/偏偏”或直接重写。
6. “就在这时”只用于真正有戏剧性打断或新事件插入；普通动作接续不要滥用。
7. “那一瞬间/一瞬间/这一刻”不要机械对应 `瞬間`。很多时候中文可直接写动作：“我反射性地蹲了下去。”
8. 原文有 deliberate repetition 时可以保留重复，但要在 `qa/<chapter>.qa.md` 说明它是节奏选择，不是懒译。

## 改写方法

### 1. 直接省略
原译：
> 不过，我只有一件事担心。

更自然：
> 我担心的只有一件事。

原译：
> 可是她没有回答，直接朝餐桌走去。

更自然：
> 她没有回答，径直朝餐桌走去。

### 2. 转成“仍然/还是/却/只”
原译：
> 即便如此，小紬还是停下了脚步。

更自然：
> 小紬还是停下了脚步。
> 小紬却停了下来。
> 她仍旧站在旁边，等到我能动为止。

### 3. 用动作承接
原译：
> 就在这时，她开口了。

更自然：
> 她开口了。
> 沉默持续了几秒，她终于开口。
> 餐桌那边传来椅子轻响。她开口了。

### 4. 用标点和换行承接
原译：
> 可是，经历这么多次后，我已经明白了。

更自然：
> 经历了这么多次，我也终于明白了。

原译：
> 那就是所谓的创伤。
> 就在这时——

更自然：
> 那大概就是所谓的创伤。
>
> 尖锐的喇叭声再次刺了过来。

### 5. 换成更轻的中文连接
可选但不要机械替换：
- 可是 → 可 / 但 / 偏偏 / 只是 / 结果 / 到头来 / 说到底 / 反倒
- 不过 → 只是 / 话说回来 / 但 / 至少 / 换句话说 / 说是这么说
- 即便如此 → 他还是 / 仍然 / 到最后 / 话虽如此 / 可她并没有……
- 就在这时 → 这时 / 下一秒 / 忽然 / 正要……时 / 直接删掉
- 那一瞬间 → 下一秒 / 刹那间 / 几乎同时 / 直接接动作

## 中文-only 通读检查
每章通读时做一次“连接词消重”：
- 标出所有段首“可是/不过/但是/即便如此/就在这时/那一瞬间”。
- 对每一处问：没有它，读者是否仍能懂前后关系？能懂就删。
- 对相邻两段都以连接词开头的地方，至少改掉一处。
- 对同一页/同一屏出现三次以上的同类转折，改成动作承接或语气承接。
- 不要用“然而/与此同时”硬替换，它们容易变成另一种翻译腔。

## 例子
偏模板：
> 可是，经历这么多次后，我已经明白了。毫无疑问，是那天的事故造成的。

较自然：
> 经历了这么多次，我也终于明白了。毫无疑问，一切都源于那天的事故。

偏模板：
> 即便如此，小紬还是停下了脚步，在我能动之前，一直待在旁边。

较自然：
> 小紬还是停了下来。在我重新动起来之前，她一直站在旁边。

偏模板：
> 不过，我只有一件事担心。那不是小紬的心情，也不是距离感，而是对我自己的不安。

较自然：
> 我担心的只有一件事。不是小紬的心情，也不是该怎么和她相处，而是我自己。
