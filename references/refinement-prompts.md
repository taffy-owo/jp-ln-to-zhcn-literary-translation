# 专项审校提示

## 译前歧义解析
```text
请只做译前解析，不要翻译。
对以下日文段落，逐段输出：
- 说话人与受话人
- 隐含主语及补出理由
- 敬语功能（礼貌/距离/亲昵/愤怒？）
- 情绪/语气功能（安慰/试探/撒娇/逞强/自嘲？）
- 信息焦点
- 歧义点与最可能解释
- 风险等级（low/medium/high）

对 high 风险的歧义点，给出两种可能解释及各自依据。
```

## 准确性审校
```text
只检查这一章的高风险准确性问题，不润色文风。
重点：说话人、指代链、动作先后、因果、漏译、反译、过译、伏笔提前暴露。
输出：critical / major / minor，并给最小修改方案。
```

## 中文-only 去翻译腔（三轨编辑）
```text
把本章当成中文小说通读，只处理不像中文小说的地方，不改变剧情事实。
分三轨依次处理：

第一轨·旁白：去说明腔。把抽象情绪名词落到动作和身体反应，删除不必要的解释性修饰。
第二轨·内心独白：保留刺感。内心话可以更尖、更软弱、更诚实，不要磨平真实想法的锐度。
第三轨·对话：维持角色差异。每句台词默读，如果听不出是谁说的，就需要调整。

输出：修订后的完整文本，每处修改用 [轨道标签] 标注。
```

## 角色声音检查（对照声线卡）
```text
只检查【角色名】在本章里的说话方式是否稳定。
对照声线卡检查以下维度：
- 称呼矩阵：对不同对象的称呼是否一致？
- 句法特征：句长、语气词偏好、口语强度是否偏离默认值？
- 情绪表达：情绪失控时的语言变化是否与性格标签一致？
- 回避表达：是否出现了这个角色绝对不会说的话？
输出：问题句 + 更贴合角色的改写 + 声线漂移严重度（minor/major）。
```

## 角色声线抽检（无声线卡时）
```text
只检查【角色名】在本章里的说话方式是否稳定。
关注：年龄感、礼貌层级、冷淡/活泼/毒舌/软弱等气质、对不同对象的称呼。
请基于已有译文，抽取该角色的声线特征：句长、语气词、回避表达、称呼习惯、情绪失控时的语言变化。
输出：问题句 + 更贴合角色的改写。
```

## 高风险词检查
```text
只检查这些表达是否误用：社会人、接住（抽象义）、违和感、元气、重要的事物、果然如此、说不定、心情复杂地、全部都由我、身体擅自、鲜明地重放、距离感。
命中时给出更自然 zh-CN 改写。
```

## 文化负载词审校
```text
只检查本章中文化负载词的处理是否一致且合理。
对照 annotation-policy.md 的五类分类规则，检查：
- 设定词是否保留了类型辨识度？
- 学校/敬称是否使用了中文等效替换？
- 双关/笑话是否保留了喜剧效果？
- 食物/日常是否过度解释或过度本土化？
- 礼俗是否在需要时做了文内轻解释？
输出：处理不一致或不合理的词 + 建议改法 + 理由。
```

## 分块合并/连贯性检查
```text
检查这一章译文是否像由多个 chunk 错误拼接而成。
重点：场景突然跳跃、时间线断裂、上一段没有承接下一段、重复段、疑似漏段、人物位置突然改变。
输出：位置、问题类型、建议修复方式。不要只做润色。
```

## 跨卷连续性回归
```text
对比当前卷与前卷的 glossary、series_bible、continuity_notes 和实际译文，找出人名/称呼漂移、口头禅漂移、固定梗漂移、世界观术语变化、关系线矛盾。
只列重大问题与统一方案。
```

## 连接词与节奏去重

```text
请只检查这一章的连接词、转折词和时间推进词是否过密，不要改剧情事实。
重点检查：
- 段首反复出现"不过/可是/然而/即便如此"
- 高频出现"就在这时/那一瞬间/下一刻/不知什么时候"
- 原文没有强转折却被译文补出转折
- 可以用句号、动作、停顿、视线、心理焦点自然承接的地方
输出：直接给出修订后的中文文本；只保留真正必要的转折词。
```

## Golden Set 回归对照
```text
请将当前译文与批准样张逐句比较，只标出以下偏离：
- 事实偏差：剧情事实、说话人归属、因果是否改变
- 称呼偏差：称呼是否与样张不一致
- 声线偏差：角色说话方式是否漂移
- 节奏偏差：停顿、留白、情绪密度是否变化
- 亲密细节偏差：感官/身体描写是否被弱化或删节

对每处偏离判定：🟢进步 / 🟡等价 / 🔴退化
输出汇总：总体判定 + 退化项列表。
```
