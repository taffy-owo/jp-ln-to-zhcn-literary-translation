# Project Memory Templates

Create these files under `meta/` for long-form projects.

## meta/series_bible.md

```md
# Series Bible

- Title:

- Genre:
- POV:
- Target reader:

## Characters
| Name | Role | First appearance | Voice | Address forms | Notes |
|---|---|---|---|---|---|

## Relationships
| Pair | Relationship | Current state | Address pattern | Change history |
|---|---|---|---|---|

## Timeline
| Chapter | Event | Continuity note |
|---|---|---|

## Motifs / Repeated Lines
| Source | zh-CN handling | Note |
|---|---|---|
```
